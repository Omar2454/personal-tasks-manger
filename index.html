Features

Task Management

Easily add, edit, and delete tasks to keep track of your personal projects.

Task Categorization

Organize your tasks by categories to streamline your workflow.

Status Tracking

Keep track of the status of your tasks, from incomplete to complete.

About

This project was inspired by the need to manage personal tasks effectively. As developers, we often have many tasks to juggle, and a simple, user-friendly task manager can help keep everything organized. This project is a Portfolio Project for Holberton School, created by User and Omar Ahmed. You can find more about our work on our LinkedIn, GitHub, and Twitter profiles linked below.
